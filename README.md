# Planning Poker ♠

A lightweight, real-time planning poker app for agile estimation. No frameworks, no build tools — just vanilla JS + Firebase.

[Live Demo](https://pawangujral.github.io/pokerface/)

## Features

- Create/join sessions via shareable link
- Fibonacci estimation scale (0, 1, 2, 3, 5, 8, 13, 21, ?, ☕)
- Real-time participant sync
- Hidden votes until "Show Votes" is clicked
- Average & consensus indicators after reveal
- Auto-cleanup of sessions after 24 hours
